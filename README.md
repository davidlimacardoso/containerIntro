# ABOUT THIS PROJECT

This project is a DevOps training, making a infraestructure and containerized applications, contains the resources below:

- RabbitMQ
- MySQL
- Memcache
- Tomcat

# Vagrant Commands

````
vagrant global-status --prune
vagrant up
vagrant ssh
vagrant destroy
````
